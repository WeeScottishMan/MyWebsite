Windows Software

Below is a list of useful windows software sorted into catagories. All software should work on Windows 10 and Windows 11 Unless directly specified.

Please let me know if you have any issues.

Performance

Privacy.sexy

A free tool/website that optimizes the security and privacy of windows as well as reducing active programs.

CTT Winutil

A free tool that debloats windows with various tweaks and a program installer. Recommend using the tweaks tab with the standard preset.


Browser Extensions

UBlock Origin

A free browser extension that is by far the best ad blocker. Make sure to check the settings if you want to enable/add your own add filters.

Utilties

StartAllBack

A paid tool (5$) with a 100 day free trail. Greatly improves Windows 11's UI, the start menu, taskbar, file explorer and dark mode are all improved.

NVCleanstall

A free tool that debloats and installs nvidia drivers allowing you to only install what you need. A must for nvidia users. (Shadowplay works with this)


Browser Extensions

Sponserblock

This is an ad block for youtube sponsorships. It simply skips the section of the video where a sponsor is placed. (No performance gains by doing this).
